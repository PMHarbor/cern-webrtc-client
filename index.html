CERN WebRTC Vidyo Client

This application is a Vidyo WebRTC Client. It is connected to the Vidyo infraestructure and allows people to make multi user videoconference calls. It is built using Javascript, Ember-Cli and Vidyo technologies.

Compatibility

  • Firefox >= 45
  • Chrome >= 50

Features

  • Multi user videoconferencing with video and audio support.
  • Desktop notifications.
  • Public and private chats.
  • Audio/video device selection.
  • Oauth2 authentication.
  • Support for private and public meetings.
  • Screen sharing (using Firefox and Chrome plugins).

Quick guide

If you don't have experience using Ember-cli, go to the "Tutorials" section.

Prerequisites

  • node >= 4.2.4
  • npm >= 3.10.5
  • bower >= 1.8.0
  • phantomjs >= 2.1.1

Build the application

npm install && bower install

Generate Self-signed certificates

  1. openssl genrsa -des3 -out server.key 2048
  2. openssl rsa -in server.key -out server.key.insecure
  3. mv server.key server.key.secure
  4. mv server.key.insecure server.key
  5. openssl req -new -key server.key -out server.csr
  6. openssl x509 -req -days 365 -in server.csr -signkey server.key -out server.crt

Move the generated files to the ssl folder.

Setup your configuration

Create a config/.env file using config/.env.sample as template.

Set up your Vidyo WebRTC plugin files

Copy your Vidyo plugin files to vendor/vidyo.

Run the application

ember s -ssl=true

Test the application

ember test --server

Build for production and qa

QA

npm run build_qa

Production

npm run build_production

The generated files will be available in dist/qa and dist/production.

Configuration

ROOT_URL=/ Path of the project SENTRY_DSN If using Sentry for logging, set it's DSN SENTRY_DEVELOPMENT Set the development mode for Sentry (true|false)

SENTRY_DEBUG Set the debug mode for Sentry (true|false)

SENTRY_CDN=https://cdn.ravenjs.com/1.2.0/raven.min.js CDN Url for the Sentry Javascript plugin.

BACKEND_SERVER_URL URL of the API and authentication backends.

OAUTH2_API_KEY Oauth ID.

OAUTH2_CALLBACK_URL Oauth callback for authentication.

PIWIK_ID Piwik ID if using Piwik for analytics.

PIWIK_URL Piwik URL if using Piwik for analytics.

SESSION_MANAGER URL of the Vidyo WebRTC session manager.

VIDYO_PORTAL_URL URL of the Vidyo Portal to generate the room URLs.

FEEDBACK_URL URL to redirect to on the feedback button.
